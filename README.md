## Overview

This repository contains my implementation of the **National Student Data Corps (NSDC) Data Science Project: Auditing AI in Criminal Justice**.

Artificial intelligence is becoming increasingly common in criminal justice systems, where machine learning models are used to support decisions such as risk assessment, recidivism prediction, sentencing recommendations, and resource allocation. While these technologies have the potential to improve efficiency, they also raise important questions about fairness, transparency, and bias.

The purpose of this project was to explore how machine learning models are built, evaluate their predictive performance, and audit their behavior using fairness metrics to better understand the ethical challenges of deploying AI in high-impact decision-making environments.

# Key Takeaways

Working on this project helped me recognize that evaluating an AI system extends beyond measuring predictive accuracy. Responsible machine learning requires understanding how data is collected, identifying potential sources of bias, evaluating fairness across different populations, and communicating results transparently.

By combining data analysis, machine learning, and ethical evaluation, I gained practical experience developing AI systems with greater awareness of their societal impact.

# What I Learned

This project reinforced several important concepts in data science and machine learning:

* Data quality has a significant impact on model performance.
* Machine learning models can unintentionally inherit historical biases from their training data.
* Accuracy alone is not sufficient when evaluating models used in high-impact domains.
* Fairness metrics provide additional insight into how models affect different populations.
* Responsible AI requires balancing technical performance with ethical considerations.
